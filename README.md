# Loopda

## Setting up locally

1. Clone the repository:
```
git clone https://github.com/quxbaz/loopda
```


2. Navigate to the directory:
```
cd loopda
```


3. Install dependencies:
```
npm install
```


4. Run the local test server:
```
npm run serve -- --port=8000
```


5. Open your browser and go to the url:
```
localhost:8000
```
